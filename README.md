1. Загальна інформація
  Даний репозиторій містить веб-застосунок, розроблений з використанням фреймворку Django мовою програмування Python.
  Проєкт реалізує серверну частину веб-системи з підтримкою маршрутизації, шаблонів, роботи з базою даних та адміністративної панелі.
  Проєкт створений відповідно до принципів MVC (Model–View–Controller), де в Django використовується архітектура MVT (Model–View–Template).

2. Вимоги до середовища
  Для коректної роботи проєкту необхідно:
  Python 3.9+
  pip (менеджер пакетів Python)
  virtualenv (рекомендовано)
  операційна система: Windows / Linux / macOS

3. Встановлення заложностей
  pip install django

4. Міграції бази даних
  python manage.py makemigrations
  python manage.py migrate

5. Створення суперкористувача
  python manage.py createsuperuser

6. Запуск сервера розробки
  python manage.py runserver
